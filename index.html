Brace Yourself!

This Ruby on Rails project provides the user portals for a smart medical device. The data from the device syncs to a database via bluetooth and this site processes and displays graphs and charts, for doctors and patients to track usage. D3.js is the data visualization library used and node.js was used as a backend data processing engine.
